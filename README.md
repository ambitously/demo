## ⚡ 30分钟快速上手

如果你只有30分钟，按这个顺序：

1. **阅读 `docs/00-题目深度分析与行动方案.md` 的「一、题目核心要素提取」和「1.2黑盒vs白盒」**（5分钟）
2. **阅读 `docs/03-Ollama环境搭建完全指南.md` 的「一、什么是Ollama」和「二、在Windows上安装」**（10分钟）
3. **在本地装好Ollama，拉取模型**（10分钟）
4. **运行 `python code/test_ollama.py` 验证**（5分钟）
