## 一、Ollama & 大模型部署

### 1.1 Ollama 官方文档
- **标题：** Ollama Official Documentation
- **URL：** https://docs.ollama.com/
- **描述：** Ollama 官方文档，包含快速开始、API 参考、模型库、工具调用等完整内容。适合所有需要本地部署大模型的场景。支持 macOS / Windows / Linux。
- **难度：** 入门
- **预计用时：** 2-4 小时
- **竞赛用途：** 本地部署 LLM 的核心工具，可直接运行 Qwen、Llama、DeepSeek 等模型

### 1.2 Ollama 官方博客

- **URL：** https://ollama.com/blog

- **难度：** 入门-进阶
- **预计用时：** 2 小时（选读重点文章）
- **竞赛用途：** 了解本地模型的最新能力边界，如 function calling 与工具调用

### 1.3 Ollama GitHub 仓库
- **标题：** ollama/ollama
- **URL：** https://github.com/ollama/ollama
- **描述：** Ollama 源代码仓库，包含社区库列表（20+ 语言支持）、Issue 讨论、贡献指南。遇到部署问题时最好的排查参考。

- **预计用时：** 1-2 小时（浏览 README 和 Wiki）

## 二、MuJoCo 仿真基础

### 2.1 MuJoCo 官方文档
- **标题：** MuJoCo Documentation
- **URL：** https://mujoco.readthedocs.io/en/stable/
- **描述：** Google DeepMind 维护的 MuJoCo 物理引擎官方文档。覆盖概述、建模语言（MJCF/URDF）、编程接口（C/Python API）、可视化、MJX（GPU 加速）、MuJoCo Warp 等内容。提供完整代码示例。
- **难度：** 入门-高级
- **预计用时：** 入门部分 4-6 小时，完整学习 20+ 小时
- **竞赛用途：** RocoBench / 多机器人协同仿真底层引擎，必学基础

### 2.2 MuJoCo GitHub 仓库
- **标题：** google-deepmind/mujoco
- **URL：** https://github.com/google-deepmind/mujoco
- **描述：** MuJoCo 开源代码仓库，包含示例模型（humanoid、robot arm 等）、C/Python 代码示例 `simulate.cc`、编译指南。模型库中有标准人形机器人和多关节机械臂模板。
- **难度：** 进阶
- **预计用时：** 3-5 小时（浏览 sample 和 model 目录）
- **竞赛用途：** 理解底层仿真代码、自定义机器人模型

### 2.3 MuJoCo Python 绑定文档
- **标题：** MuJoCo Python Bindings
- **URL：** https://mujoco.readthedocs.io/en/stable/python.html
- **描述：** MuJoCo 的 Python 接口，通过 `pip install mujoco` 即可使用。提供与 C API 完全对应的 Python 封装，支持 `dm_control` 等上层框架集成。
- **难度：** 入门
- **预计用时：** 2-3 小时
- **竞赛用途：** Python 环境下的机器人仿真编程

### 2.4 dm_control - DeepMind 控制套件
- **标题：** DeepMind dm_control
- **URL：** https://github.com/google-deepmind/dm_control
- **描述：** DeepMind 基于 MuJoCo 构建的强化学习与控制套件，提供丰富的预置环境和评估基准。是使用 MuJoCo 进行 RL 研究的标准工具。
- **难度：** 进阶
- **预计用时：** 4-6 小时
- **竞赛用途：** 标准化的 RL 训练环境，可迁移经验到 RocoBench

### 2.5 B站：MuJoCo 入门系列教程
- **标题：** MuJoCo 仿真入门到实战（B站搜索推荐）
- **URL：** https://search.bilibili.com/all?keyword=MuJoCo仿真入门&order=click
- **描述：** B站上 MuJoCo 相关中文视频教程，按播放量排序，涵盖安装配置、MJCF 建模、Python 交互等。搜索关键词："MuJoCo仿真入门"、"MuJoCo强化学习"。
